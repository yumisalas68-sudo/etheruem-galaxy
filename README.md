# 🌌 Ethereum Ecosystem Galaxy

An immersive, highly performant interactive 3D WebGL data visualization platform showcasing the vastness of the Ethereum Ecosystem. Rendered dynamically via Three.js, it securely tracks and orbits over 1,700 active Ethereum projects and tokens in a fluid, interactive digital universe.

live website ; https://yumisalas68-sudo.github.io/etheruem-galaxy/



---

## 🚀 Features

- **Massive Scale Rendering**: Utilizes Three.js `InstancedMesh` logic to render and continuously animate 1,722 individual project data nodes at 60FPS without frame dropping.
- **Bimodal Theme Engine**:
  - **Dark Space**: Deep contrast, high-bloom emissive tracking for classic crypto cyberpunk aesthetics.
  - **Ethereum Silk**: A fully inverted, ultra-chic `#F0F2F9` silvery-blue mode showcasing high-fidelity material adjustments.
- **Raycast Interaction Engine**: Fluid "Hover-to-Inspect" event listeners tied to dynamic camera vectors globally tracking intersecting coordinates.
- **Instant Search Subsystem**: Quickly filter and teleport to specific protocols across 35 distinct algorithmic orbital rings.

## 🛠 Tech Stack

- **Framework:** React 19 / Vite
- **Graphics Engine:** Three.js / `@react-three/fiber`
- **Routing:** TanStack Router (Base-path optimized for GH Pages CI/CD)
- **Styling:** Tailwind CSS V4 + Radix UI Primitives
- **Data Architecture:** CSV Data Ingestion Module via `papaparse` 
- **Validation/Testing:** TestSprite Advanced Agentic Cloud Backend
